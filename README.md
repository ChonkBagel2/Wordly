## Steps to run - 

1) Clone this repo
2) Run the following commands
3) ```
   npm install
   nodemon index.js
   ```
4) Go to localhost:3000/words  


## Tech Stack - 

Node.js, Express, EJS, HTML, CSS, JavaScript
